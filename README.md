# 🎵 Tamil Beats — MP3 Player

A beautiful Tamil music player with category filtering, built with pure HTML, CSS & JavaScript.

## 📁 Project Structure

```
tamil-player/
├── index.html        ← Main page
├── css/
│   └── style.css     ← All styles
├── js/
│   └── app.js        ← All logic
├── songs/            ← (Optional) Put MP3 files here
│   ├── melody/
│   ├── beat/
│   ├── breakup/
│   ├── love/
│   ├── mass/
│   └── devotional/
└── README.md
```

## 🚀 How to Run Locally (VS Code)

### Option 1 — Live Server (Recommended)
1. Open the `tamil-player` folder in VS Code
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **Open with Live Server**
4. Browser opens automatically at `http://localhost:5500`

### Option 2 — Just double-click
- Double-click `index.html` directly — opens in your browser instantly!

## 🎶 How to Use

1. Click **+ Add Songs** button (top right)
2. Select your Tamil MP3 files
3. Choose a category (Melody, Beat, Break Up, Love, Mass, Devotional)
4. Songs appear in the list — click to play!

### You can also:
- **Drag & Drop** MP3 files anywhere on the page
- **Search** songs by name
- Filter by **category** from the sidebar
- Use **keyboard shortcuts**:
  - `Space` → Play / Pause
  - `→` / `←` → Skip 5 seconds
  - `↑` / `↓` → Volume up / down
  - `N` → Next song
  - `P` → Previous song

## ☁️ Deploy to Render

1. Push this folder to a **GitHub repository**
2. Go to [render.com](https://render.com) and sign in
3. Click **New → Static Site**
4. Connect your GitHub repo
5. Set:
   - **Build Command**: *(leave empty)*
   - **Publish Directory**: `.` (dot)
6. Click **Deploy** — done! 🎉

Your site will be live at `https://your-site-name.onrender.com`
